Analysis of Apartment Rentals in Jacksonville, Florida

In this project, I scraped real estate listings from Apartments.com for the Jacksonville area using Selenium and BeautifulSoup. After collecting the data, I cleaned and processed it, extracting geospatial information with Geopy's ArcGIS. To visualize key trends, I created scatter plots with different hues representing variables such as price, number of bedrooms, and square footage. I then applied K-means clustering to group the apartments into distinct Jacksonville neighborhoods and labeled them accordingly. To investigate price differences between these clusters, I conducted an ANOVA test followed by post-hoc analysis. I also used KDE plots to examine apartment density across the city and within individual neighborhoods. Lastly, I built predictive models to estimate rental prices based on square footage and coordinates, testing several modeling approaches. The project concluded with insights on where the average citizen of this area might find affordable housing in Jacksonville.

Predicting Specific Gravity in Potatoes Using Near-Infrared Spectroscopy

This project focused on predicting the specific gravity of potatoes, which is closely related to starch content, using Near-Infrared (NIR) spectroscopy data. Specific gravity is a key quality indicator in potatoes, traditionally measured through labor-intensive methods. To automate this process, I built a machine learning pipeline to analyze the spectral data and estimate specific gravity values.Key steps included preprocessing the NIR data using Savitzky-Golay derivatives and Multiple Scatter Correction (MSC) to reduce noise and improve signal quality. For feature reduction, I applied Principal Component Analysis (PCA) and K-means clustering to group similar spectral profiles. The predictive model was built using Partial Least Squares (PLS) regression, which was evaluated through cross-validation to ensure robustness.

Automated Image-Based Recognition of Orange Tree ID Tags Using CNNs and OCR

In this project, I developed an automated system to recognize and catalog ID tags on orange trees using Convolutional Neural Networks (CNNs) and Optical Character Recognition (OCR). The main challenge was to create an efficient, scalable solution for identifying small, often weathered tags amidst complex outdoor environments. By leveraging a VGG16 architecture, pre-trained on ImageNet and fine-tuned for the task, I was able to achieve high accuracy in image classification.

Automated Image-Based Scoring of Citrus Tree Health Using Regression CNNs

In this project, I developed a regression Convolutional Neural Network (CNN) model to visually score the health of citrus trees from images on a scale of 1-20. Higher scores represent trees severely affected by Huanglongbing (HLB), while lower scores indicate healthier trees with minimal disease progression. The model was trained on human-assessed scores, previously done manually in the lab, making the process highly labor-intensive. By automating this task, I drastically reduced the time required for evaluation and improved the consistency of disease assessment.
